jQuery.remainingCharacters

This page shows the remainingCharacters plugin running in various configurations.

This is the way it works with minimum configuration.

<div id="output" class="target"<>/div>
<input id="textbox" style="width: 100%;" />
<script type="text/javascript">
	$().ready(function(){
		$('#textbox').remainingCharacters({
			maxChars: 140
		});
</script>

This shows how the plugin works with the optional parameters set.

  • hideTarget: true
  • fadeTarget:
    • fade: true
    • speed: slow

The output is hidden until the input box has focus, at which point it fades in, and it fades out when the textbox loses focus.

<div id="output2" class="target"></div>
<input id="textbox2" style="width: 100%;" />
<script type="text/javascript">
	$().ready(function(){
		$('#textbox2').remainingCharacters({
			target: $('#output2'),
			maxChars: 140,
			hideTarget: true,
			fadeTarget: {
				fade: true,
				speed: 'slow'
			}
		});
</script>

This shows how the plugin works with the optional parameters set and the output rendered at run time in a span instead of being specified at design time.

  • hideTarget: true
  • fadeTarget:
    • fade: true
    • speed: slow

The output is hidden until the input box has focus, at which point it fades in, and it fades out when the textbox loses focus.

This shows how the plugin works with the optional parameters set and the output rendered at run time in a span instead of being specified at design time.

  • hideTarget: true
  • fadeTarget:
    • fade: true
    • speed: slow

The output is hidden until the input box has focus, at which point it fades in, and it fades out when the textbox loses focus. This one also uses the new targetStyle introduced in version 2.3.0 to allow the default target rendered at runtime to be styled.

<input id="textbox4" style="width: 100%;" />
<script type="text/javascript">
	$().ready(function(){
		$('#textbox4').remainingCharacters({
			maxChars: 140,
			hideTarget: true,
			fadeTarget: {
				fade: true,
				speed: 'slow'
			},
			targetStyle: 'width: 100%; text-align: left; font-weight: bold; color: red;'
		});
</script>
