jQuery.remainingCharacters

The easiest way to install the plugin is to use NuGet. In Visual Studio, simply use the following command in the Package Manager Console.

Install-Package jQuery.remainingCharacters

This is the way it works with minimum configuration.  The only truly mandatory configuration property is  the maximum number of characters.

<div id="output" class="target"<>/div>
<input id="textbox" style="width: 100%;" />
<script type="text/javascript">
	$().ready(function(){
		$('#textbox').remainingCharacters({
			maxChars: 140
		});
</script>

This shows how the plugin works with some of the optional parameters set.  Setting hideTarget to true makes the target element hidden by default until the textbox receives focus.  The fadeTarget is a javascript object that contains two sub-properties, fade and speed.  Set the fade property to true to enable the jquery.fade transition.  The speed property is self-explanatory.  The target property is only required if you want to specify a specific html element to receive the number of characters remaining.

<div id="output2" class="target"></div>
<input id="textbox2" style="width: 100%;" />
<script type="text/javascript">
	$().ready(function(){
		$('#textbox2').remainingCharacters({
			target: $('#output2'),
			maxChars: 140,
			hideTarget: true,
			fadeTarget: {speed: 'slow'}
		});
</script>

This shows how the plugin works with the optional parameters set and the output rendered at run time in a <div> instead of being specified at design time and hidden until the textbox gets focus.  It is also set of fade in and out slowly.

<input id="textbox3" style="width: 100%;" />
<script type="text/javascript">
	$().ready(function(){
		$('#textbox3').remainingCharacters({
			maxChars: 140,
			hideTarget: true,
			fadeTarget: {speed: 'slow'}
	});
</script>

This shows how the plugin works with the optional parameters set and the output rendered at run time in a span instead of being specified at design time.  The only difference between this on and the previous example is that it also uses the targetStyle property to specify the css styles to be applied to the <div> that is rendered at runtime.

<input id="textbox4" style="width: 100%;" />
<script type="text/javascript">
	$().ready(function(){
		$('#textbox4').remainingCharacters({
			maxChars: 140,
			hideTarget: true,
			fadeTarget: {speed: 'fast'},// speed: fast the the default so this will also work fadeTarget: {}
			targetStyle: 'width: 100%; text-align: left; font-weight: bold; color: red;'
		});
</script>
